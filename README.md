# 🧮 Super Calculator v3.0.0 ✨

[![Telegram](https://img.shields.io/badge/Telegram-Channel-blue?style=for-the-badge&logo=telegram)](https://t.me/JoeMoeCode)

Современный калькулятор на базе **Flet**, объединяющий мощь инженерных вычислений, удобство умного конвертера и уникальную систему "Живого отклика". Создан не только для математики, но и для того, чтобы удивлять!

---

## 🚀 Ключевые возможности

### 💎 Умный конвертер
- **Валюты**: Актуальные курсы (RUB, USD, EUR, GEL, KZT) с онлайн-обновлением и кэшированием.
- **Величины**: Вес, Длина, Температура, Скорость.
- **Swap**: Кнопка мгновенного обмена единиц From/To.

### 🧬 Инженерный режим
- Полный набор функций: `sin`, `cos`, `tan`, `log`, `ln`, `sqrt`, `x²`, `x³`, `n!`.
- Постоянные: `π`, `e`.
- Память: Кнопка `Ans` для быстрого доступа к последнему результату.

### ⌨️ Keyboard Support
- Полноценный ввод с клавиатуры.
- Ввод секретных слов для активации пасхалок.
